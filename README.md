# raft

[![Build Status](https://travis-ci.com/ylgrgyq/raft.svg?branch=master)](https://travis-ci.com/ylgrgyq/raft)

Write a raft to know more about it.

Supported raft features:
- Leader election
- Log replication/compaction
- Membership changes
- Leadership transfer
- Pipeline
- Snapshot
